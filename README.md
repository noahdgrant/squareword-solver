# Squareword Solver

This project is a solver for the [Squareword](https://squareword.org/) game.

## Build

Build the project with...

```bash
$ make
```

## Run

Run the binary from the project root with...

```bash
$ ./bin/main
```

## Test

Run the tests with...

```bash
$ make test
```

## Clean

Clean old build files with...

```bash
$ make clean
```
